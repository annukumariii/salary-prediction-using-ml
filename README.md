 Salary Prediction using Machine Learning
 
 This project predicts salary based on years of experience using Simple Linear Regression.


 Dataset

The dataset is sourced from Kaggle and contains:

* YearsExperience
* Salary



 Current Progress

* ✅ Simple Linear Regression implemented
* ⏳ Multiple Linear Regression coming soon
